Cube snake or snake cube is a puzzle game. In this, all the cubelets
are tied together in a thread. Some of the cubelets are are connected
to each other, while some are not.

Amazon link for a this product:-

Wikipedia:-
https://en.wikipedia.org/wiki/Snake_cube

While each cubelet can be rotated along with all the other cubelets
that are connected with it, the aim is to put together all the cubelets in a
sequence to create a 3x3 cube out of it.
